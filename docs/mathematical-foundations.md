# 记忆系统的数学基础

本文档整理了可用数学方式形式化表示的记忆系统模型，按数学表达力从具体到抽象组织，并标注与 Engram/Memento 设计的对应关系。

---

## 一、遗忘曲线的数学模型

### 1.1 旧版公式（指数衰减）

```
effective_strength = strength × 0.5^(hours / half_life)
```

经典的指数衰减模型。优点是简单，但其记忆强度随时间衰减过快，不完全符合人类长时记忆的“长尾”特性——即记忆在很长一段时间后仍会保留微弱强度。

### 1.2 当前公式（简化幂律衰减）

为解决指数衰减的“长尾”问题，当前实现采用了一个受 FSRS v6 启发的简化幂律模型：

```
effective_strength = strength × (1 + hours_since / half_life) ^ -0.5
```

该模型：
- **保留了稳定性计算**：复用了原有的 `half_life` 计算逻辑作为核心稳定性参数 `H` (公式中的 `half_life`)。
- **引入了幂律衰减**：将衰减曲线从 `c^t` 的指数形式替换为 `t^c` 的幂律形式。
- **具有长尾效应**：相比指数模型，它在长时间段（例如数周或数月）后能保留更高的记忆强度，更符合记忆的实证研究。

这个公式可以看作是 FSRS v6 幂律模型在一个关键参数（`DECAY ≈ -0.5`）下的简化应用，在工程上实现了更好的记忆模拟效果和计算效率之间的平衡。

### 1.3 幂律衰减理论

**来源**: Wixted & Carpenter 2007; Kahana & Adler 2017

```
P(recall) = m · (1 + h·t)^{-f}
```

Kahana 证明了一个优美的数学事实：如果每条记忆的衰减率 λ 服从 Gamma 分布，那么聚合遗忘曲线就是幂律：

```
若单条记忆: R = e^{-λt}, λ ~ Gamma(α, β)
则群体平均: R̄(t) = (β / (β + t))^α    ← 幂律
```

**与 Engram 的关联**：记忆库里不同记忆有不同 `decay_rate`，天然满足"不同衰减率"的前提。聚合检索质量应表现为幂律而非指数衰减。

### 1.3 FSRS v6 的参数化幂律

**来源**: Ye et al., KDD 2022

```
R(t, S) = (1 + F · t/S)^{-1/w₂₀}
```

其中 `F = 19/81`（由约束 `R(S,S) = 0.9` 推导），`w₂₀` 是可学习参数（控制曲线形状）。

稳定性更新（成功回忆后）：

```
S' = S · (1 + e^{w₈} · (11-D) · S^{-w₉} · (e^{w₁₀(1-R)} - 1))
```

关键洞察：

- `S^{-w₉}` — **稳定性饱和**：越稳定的记忆越难进一步增强
- `e^{w₁₀(1-R)} - 1` — **间隔效应**：越接近遗忘时成功回忆，增益越大

与现有 `reinforcement_boost = min(0.1, 0.05 × (1 + log(1 + hours)))` 在形式上相似，但 FSRS 的参数化更精确。

## 二、ACT-R 激活方程

**来源**: Anderson & Lebiere 1998; Stocco et al. 2023

认知科学中最广泛验证的记忆数学模型。

### 总激活量

```
Aᵢ = Bᵢ + Sᵢ + ε
```

| 分量 | 公式 | 含义 | Engram 对应 |
|------|------|------|-------------|
| Bᵢ (基线激活) | `ln(Σⱼ tⱼ^{-d})` | 频率 + 近因性 | `effective_strength` |
| Sᵢ (扩散激活) | `(W/N) · Σ S_{q→i}` | 上下文相关性 | `similarity_score` |
| ε (噪声) | `Logistic(0, s)` | 随机涟漪 | 暂未实现 |

### 基线学习方程（BLL）

```
Bᵢ = ln( Σⱼ₌₁ⁿ tⱼ^{-d} )
```

其中 `tⱼ` 是第 j 次访问距今的时间，`d ≈ 0.5` 是衰减指数。

近似形式（适合工程实现）：

```
Bᵢ ≈ ln( n / (1-d) · L^{-d} )
   = ln(n) - d · ln(L) + const
   = log(频率) - d × log(寿命)
```

### 检索概率

```
P(retrieval) = 1 / (1 + e^{-(Aᵢ - τ) / s})
```

### 贝叶斯解释 (Stocco et al. 2023)

ACT-R 激活方程等价于贝叶斯后验：

```
A(m) = log P(m|Q) = log P(m) + log P(Q|m) - log P(Q)
     = B(m)        + S(m)        + const
       先验(需求概率)  似然(上下文匹配)
```

**关键启示**：`effective_strength × similarity` 排序本质上就是 `exp(B + S)` 的非对数形式。ACT-R 告诉我们这是**贝叶斯最优检索**。

## 三、Modern Hopfield Network

**来源**: Ramsauer et al., ICLR 2021; Santos et al., ICML 2024 / JMLR 2025

近年最重要的数学发现之一：**Transformer 的注意力机制 = 联想记忆的检索过程**。

### 能量函数

```
E(q) = -(1/β) · log( Σᵢ exp(β · xᵢᵀq) ) + ½‖q‖²
```

### 更新规则（= 一层注意力）

```
q^{t+1} = Xᵀ · softmax(β · X · qᵗ)
```

当 `β = 1/√D` 时，这数学上等价于一个 Transformer 注意力头。

### 存储容量

```
经典 Hopfield:  p_max ~ N        (线性于神经元数)
Modern Hopfield: p_max ~ exp(D/2)  (指数于维度!)
```

### 统一框架 — Hopfield-Fenchel-Young (Santos et al., JMLR 2025)

```
E(q) = -Ω*(X·q) + Ψ(q)
```

其中 `Ω*` 是 Fenchel 共轭。不同的 Ω 选择恢复不同的注意力变体：

| Ω | 对应的检索方式 |
|---|--------------|
| Shannon 负熵 | softmax 注意力 |
| Tsallis α-负熵 | sparsemax 注意力 |
| SparseMAP | 结构化模式关联检索 |

**关键定理**：当 Fenchel-Young 损失有正边际时，精确记忆检索成立。**稀疏性 ↔ 边际 ↔ 精确检索**。

**与 Engram 的关联**：`recall` 用向量相似度做检索，本质上是在做一步 Hopfield 能量最小化。引入 sparsemax 替代 softmax 可使检索结果更稀疏精准。

## 四、信息论记忆模型

**来源**: Gershman 2024, Oxford Handbook of Human Memory; Jakob & Gershman 2023, eLife

**核心洞察：遗忘 = 有损压缩**

### 率失真理论框架

记忆被建模为一个容量有限的通信信道：

```
min_{Q(m̂|x)} I(X; M̂)    subject to    E[d(x, x̂)] ≤ D_max
```

- `I(X; M̂)` — 记忆需要的比特数（越少越好）
- `d(x, x̂)` — 回忆与原始的失真（越小越好）
- **遗忘 = 沿率失真曲线向低比特率方向滑动**

### 最优编码的拉格朗日形式

```
L = I(X; C) + β · E[d(X, C)]
    复杂度      失真度
```

β 控制精度-效率权衡：
- β 小 → 记住要点（语义化）
- β 大 → 逐字记忆（情景）

**与 Engram 的关联**：`episodic → semantic` 抽象化过程，数学上就是沿率失真曲线从高比特率（情景细节）向低比特率（语义要点）的移动。`rigidity` 参数控制的是 β 值——高刚性 = 大 β = 不允许压缩。

### ICLR 2026 — "Learning is Forgetting"

Conklin et al. 证明 LLM 训练本身就是最优有损压缩。压缩质量（通过互信息度量）可预测下游性能。

## 五、动力系统记忆模型

**来源**: Fiebig et al., PLOS Computational Biology 2024

### 神经动力学

```
τᵣ · drᵢ/dt = -rᵢ(t) + r₀ + φ(hᵢ(t), θᵢ(t)) + C·ξᵢ(t)
```

### 突触可塑性（Hebbian + 遗忘）

```
τ_w · dwᵢⱼ/dt = η · rᵢ(t) · rⱼ(t) - β · wᵢⱼ(t)
                  ───────────────────   ──────────────
                  赫布学习(共激活增强)     主动遗忘(衰减)
```

**与 Engram 的关联**：Nexus 赫布学习 `η·rᵢ·rⱼ` 对应共同激活的记忆相互增强（`coactivated_pairs`），`β·wᵢⱼ` 对应 `association_strength` 的自然衰减。这个微分方程就是 Nexus 动力学的数学基础。

## 六、代数拓扑记忆模型

**来源**: "Cycle is All You Need", arXiv 2025

用代数拓扑的同调论形式化记忆巩固：

```
边界算子: d² = 0
被遗忘的碎片 = H₀(Z) 中的平凡 0-环（孤立连通分量，无关系结构）
被巩固的持久记忆 = H₁(Z) 中的非平凡 1-环（闭合关系环路，抗扰动）
```

**直觉**：内容被编码为"点"，只有当它们通过上下文关系连接成闭合环路时，才能巩固为持久记忆。孤立的点（无关联的信息）会被遗忘。

**与 Engram 的关联**：Nexus 网络中，孤立的 Engram（无 Nexus 连接）衰减最快，而被多条 Nexus 连接成"环路"的记忆群最抗遗忘——这正是 H₁ 非平凡环路的工程实现。

## 七、CLS 的数学形式化

**来源**: McClelland et al. 1995; Sun et al., Nature Neuroscience 2023; Jain & Lindsey 2025

### 双系统权重更新

```
海马体: Δwᵢⱼ = η_fast · xᵢ · xⱼ         (赫布，高学习率)
新皮层: Δwᵢⱼ = η_slow · (tⱼ - oⱼ) · xᵢ   (误差驱动，低学习率)
```

核心约束：`η_fast >> η_slow`，海马体通过重放训练皮层（交错样本防灾难性遗忘）。

### EWC 防遗忘正则化 (Kirkpatrick et al. 2017)

```
L_total = L_new(θ) + (λ/2) · Σᵢ Fᵢ · (θᵢ - θ*ᵢ)²
```

`Fᵢ` 是 Fisher 信息矩阵对角线（参数 i 对旧任务的重要性）。重要参数被"锚定"，不重要的可以自由学新东西。

**与 Engram 的关联**：`rigidity` 本质上就是 EWC 的 `Fᵢ`——高 rigidity 的记忆（事实、配置）被锚定不可修改，低 rigidity 的（情景）可被再巩固。

## 八、协议设计的数学基础

从数学角度，一个完整的记忆协议需要形式化六个核心操作：

| 操作 | 数学工具 | 推荐模型 |
|------|---------|---------|
| **Encode**（编码） | 向量嵌入 + Hopfield 能量 | Modern Hopfield 存储 |
| **Retrieve**（检索） | ACT-R 激活方程 B + S + ε | 贝叶斯最优检索 |
| **Decay**（衰减） | FSRS 幂律 `(1 + Ft/S)^C` | 参数化遗忘曲线 |
| **Consolidate**（巩固） | 率失真压缩 `min I(X;M) s.t. E[d] ≤ D` | 信息论有损压缩 |
| **Associate**（关联） | 赫布微分方程 `dw/dt = η·rᵢrⱼ - β·w` | 动力系统吸引子 |
| **Forget**（遗忘） | 同调论 `H₀`(碎片遗忘) vs `H₁`(环路保留) | 拓扑不变量 |

最有工程可行性的组合：**ACT-R 的贝叶斯解释 + FSRS 的参数化衰减 + Hopfield-FY 的能量函数**。
